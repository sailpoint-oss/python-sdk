# CertificationPhase

The current phase of the campaign. * `STAGED`: The campaign is waiting to be activated. * `ACTIVE`: The campaign is active. * `SIGNED`: The reviewer has signed off on the campaign, and it is considered complete. 

## Enum

* `STAGED` (value: `'STAGED'`)

* `ACTIVE` (value: `'ACTIVE'`)

* `SIGNED` (value: `'SIGNED'`)
