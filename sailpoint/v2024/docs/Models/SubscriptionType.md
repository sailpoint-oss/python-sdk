# SubscriptionType

Subscription type. **NOTE** If type is EVENTBRIDGE, then eventBridgeConfig is required. If type is HTTP, then httpConfig is required.

## Enum

* `HTTP` (value: `'HTTP'`)

* `EVENTBRIDGE` (value: `'EVENTBRIDGE'`)

* `INLINE` (value: `'INLINE'`)

* `SCRIPT` (value: `'SCRIPT'`)

* `WORKFLOW` (value: `'WORKFLOW'`)
