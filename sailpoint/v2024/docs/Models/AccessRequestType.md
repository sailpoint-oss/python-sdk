# AccessRequestType

Access request type. Defaults to GRANT_ACCESS. REVOKE_ACCESS type can only have a single Identity ID in the requestedFor field.

## Enum

* `GRANT_ACCESS` (value: `'GRANT_ACCESS'`)

* `REVOKE_ACCESS` (value: `'REVOKE_ACCESS'`)
